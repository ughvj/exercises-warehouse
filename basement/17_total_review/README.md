## 概要

- 今まで講義でやった内容の総まとめです
- 問題ごとに過去の講義番号とキーワードを書いていますので、詰まった時は参考にしてみてください

## :turtle: 問題1

- 2つの数字を変数にして、加減乗除(+, -, *, /)を計算して出力しよう
- たとえば10と5の場合、以下のように出力されればOKです
- 講義番号
  - 01_variables
- キーワード
  - 変数
  - int
  - 四則演算
  - 画面出力
  - printf

出力

```
15
5
50
2
```

## :dog: 問題2

- 以下に動作例を示します
- 講義番号
  - 02_input_values
- キーワード
  - scanf

入力

```
10
5
```

出力

```
15
5
50
2
```

## :bear: 問題3

- 2つの数字を入力して、それらの加減乗除(+, -, *, /)を計算して出力しよう
- ただし、「1つ目の値 < 2つ目の値」を満たす場合、「入力が正しくありません」と出力しよう
- 以下に動作例を示します
- 講義番号
  - 03_if
- キーワード
  - if
  - else

入力

```
5
10
```

出力

```
入力が正しくありません
```

## :whale: 問題4

- 2つの数字を入力して、それらの加減乗除(+, -, *, /)を計算して出力しよう
- ただし、「1つ目の値 < 2つ目の値」を満たす場合、再度入力を受け付けよう
- 以下に動作例を示します
- 講義番号
  - 04_for
  - 05_while
- キーワード
  - for
  - while

入力

```
5
10

```

出力

```
15
5
50
```
