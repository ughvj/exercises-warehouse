## :turtle: 問題1

- 1つの数字を入力して、その数字が40以下なら "不合格"、41以上なら"合格"と出力しよう
- 以下に動作例を示します

入力

```
41
```

出力

```
合格
```

## :dog: 問題2

- 2つの数字を入力して、それらの商を出力しよう
- 1つ目の入力が割られる数、2つ目の入力が割る数とします
- ただし、割る数に0が入力された場合、"ゼロ除算です" と出力します
- 以下に動作例を示します

入力

```
10
0
```

出力

```
ゼロ除算です
```
