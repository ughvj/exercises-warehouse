## :dog: 問題2

- 2つの数字を入力して、それらの商を出力しよう
- 1つ目の入力が割られる数、2つ目の入力が割る数とします
- ただし、割る数に0が入力された場合、"ゼロ除算です" と出力します
- HINT: 2つ目の入力が0の場合に分岐します

入力

```
10
0
```

出力

```
ゼロ除算です
```
